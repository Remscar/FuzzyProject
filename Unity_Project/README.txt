To Run the project do the following:

1. Open the Application Folder
2. Double Click and run Application.exe
3. The Unity game launcher will open. Hit the Play button to launch the project.

Once the project opens you should see a car move through an obstacle course using a fuzzy solver for turning angle and speed.

Input:

Close Program: Esc

The project takes no input to affect the scene.

Source Code:

The source code for this project can be found in the following project path

MAT362-Project1\Assets\Scripts

Scripts of note for the project are:

FuzzyNumber.cs
MamdaniSolver.cs
CarController.cs (Consequence and Antecedent definitions found here)
